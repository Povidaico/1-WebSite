# 🏀 Projeto Franca Basquete - Web Development

Este é o meu primeiro projeto prático desenvolvido durante a primeira semana do curso de programação. O objetivo inicial era criar uma página simples de nivelamento em HTML e CSS, mas decidi expandir o escopo para criar um site mais completo e temático sobre o **Sesi Franca Basquete**.

## 🚀 Sobre o Projeto
O site busca reunir informações sobre a história, conquistas e ídolos do time de Franca, unindo o aprendizado técnico com a paixão pelo esporte. 

Atualmente, o projeto está em **fase de desenvolvimento (WIP - Work in Progress)**.

### ✅ O que já foi implementado:
- Estrutura semântica em HTML5.
- Organização de seções (História, Títulos e Elenco).
- Primeira versão da Home Page.

### 🛠️ Tecnologias Utilizadas:
- **HTML5**: Estruturação de conteúdo.
- **CSS3**: Estilização e layout (em progresso).
- **Git/GitHub**: Controle de versão e hospedagem.
